# Scrabber API

## Get it Running

1. To run **Scrabber**:

```bash
   $ docker build -t scraber .
   $ docker run --publish 3001:3001 scrabber-api
```

2. Commands to run:

```bash
   # development
   $ npm run start

   # watch mode
   $ npm run start:dev

   # production mode
   $ npm run start:prod
```
